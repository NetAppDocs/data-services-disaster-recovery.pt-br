= Conclusão
:allow-uri-read: 


[role="lead"]
O NetApp Disaster Recovery fornece uma solução de DR de nuvem híbrida e econômica para sua infraestrutura VMware local quando configurado para usar o Amazon EVS e o Amazon FSx for NetApp ONTAP como um destino de DR.  Isso fornece a você uma meta de DR robusta para infraestrutura de missão crítica.  O NetApp Disaster Recovery aproveita os recursos nativos de proteção de dados do ONTAP , respaldados por décadas de experiência em proteção de dados, profundamente testada para garantir que seus dados estejam totalmente protegidos.

Para mais informações:

* https://docs.aws.amazon.com/fsx/latest/ONTAPGuide/getting-started.html["Introdução ao Amazon FSx for NetApp ONTAP"^]
* https://docs.netapp.com/us-en/bluexp-disaster-recovery/index.html["Documentação do NetApp Disaster Recovery"]
